Policy quick reference
  • Rate limit — How many requests per minute an agent can make. Start lower for write operations.
  • Human consent — "Yes" means the human must approve before each use (Tier-2 Passport). "No" means the agent can act with just its own token (Tier-1).
  • Scope — The permission string checked at request time (usually service:action).
